# 摩尔斯电码学习工具

一个网页端的摩尔斯电码（Morse Code）练习工具，为业余无线电爱好者（HAM）设计。支持从入门到进阶的多种模式，帮助提升接收和发送技能。界面简洁，支持浏览器直接运行。

## 简介

本工具基于 Koch 方法，从简单字符开始逐步增加难度，保持较高速度（默认 20 WPM）。它包括字符学习、速度练习、消息训练、音频生成和参考查询。默认使用 Farnsworth 节奏（适合初学者），可自定义速度（5-60 WPM）和音调（300-1500 Hz）。练习数据本地保存，追踪你的进度。

## 功能

- **Koch 学习法**：从 K 和 M 开始，40 级课程。每级添加新字符，显示当前字符和新增字符，支持播放、输入检查和音频下载。
- **速度练习**：选择代码组、纯文本、呼号或通联简语。自定义时长（10-120 秒）和重复播放，提高准确性。
- **WAE QTC 训练**：模拟 QTC 消息交换，支持 WAE/DX 格式和 1-10 条消息练习。输入时用 / 分隔消息。
- **WAV 生成器**：创建自定义音频（Koch 课程、随机字符、通联简语或呼号），预览文本后生成并下载 WAV 文件，用于离线练习。
- **文本转换器**：输入文本转换为摩尔斯码，反之亦然。支持播放、复制代码和音频导出。
- **摩尔斯码参考表**：交互式表格，按字母/数字/符号分组。点击任意字符播放其音频。

## 快速开始

1. **运行工具**：
   - 下载文件或直接打开 `index.html` 在浏览器中（推荐 Chrome 或 Firefox，支持 Web Audio API）。
   - 无需安装，纯网页运行。首次播放音频可能需点击页面允许。

2. **导航和使用**：
   - **顶部菜单**：点击 Koch 学习法、速度练习等切换模块（移动端用汉堡菜单）。
   - **通用控件**：
     - 速度滑块：调整 WPM（显示实时值）。
     - 音调滑块：设置 Hz（默认 600 Hz）。
     - 时长滑块：控制练习/音频长度。
     - 复选框：启用 Farnsworth 节奏（降低间隙，保持字符速度）。
   - **练习流程**（以 Koch 为例）：
     1. 选择级别（1-40），查看当前/新增字符。
     2. 点击“播放”听音频，在输入框键入听到的字符。
     3. 点击“检查答案”获取反馈（正确率实时更新）。
     4. 用“重新生成练习”刷新内容，或下载音频离线用。
   - **其他模块类似**：播放 → 输入 → 检查 → 下载/再生。
   - **提示**：从 Koch 1 级开始，每天练习 30 秒。正确率 >90% 后进阶。统计在页面底部显示，总练习/正确/准确率。

3. **自定义练习**：
   - WAV 生成器：选类型（如 Koch），调整参数，点击“生成音频”下载。
   - 转换器：输入文本（如 “CQ DE BA6AA”），自动转码并播放。
   - 参考表：点击字符听音，便于记忆。

如果遇到音频问题，确保浏览器未静音，并刷新页面。
